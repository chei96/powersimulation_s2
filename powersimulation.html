Feedback and Accuaracy

The power simulation pertaining to feedback seeking and accuracy is based on two questions:

  • does the feedback increase accuracy immediately after receiving feedback?
  • does the accuracy for those who receive feedabck decreases over time to it’s initial level?

We simulated the power based on the following path model, using the r-package {r} library(lavaan)

Population model for data generation

pop_mod <- "
#effects in both groups
self_t0 ~ c(.3,.3)*iqtest
self_t1 ~ c(.3,.6)*iqtest
self_t2 ~ c(.3,.5)*iqtest
self_t3 ~ c(.3,.4)*iqtest
self_t4 ~ c(.3,.3)*iqtest

#add covariances of self-perceptions over time (irrespective of iq)
self_t0 ~~ c(.3,.3)*self_t1
self_t0 ~~ c(.3,.2)*self_t2
self_t0 ~~ c(.25,.2)*self_t3
self_t0 ~~ c(.2,.2)*self_t4
self_t1 ~~ c(.3,.3)*self_t2
self_t1 ~~ c(.25,.25)*self_t3
self_t1 ~~ c(.2,.2)*self_t4
self_t2 ~~ c(.3,.3)*self_t3
self_t2 ~~ c(.25,.25)*self_t4
self_t3 ~~ c(.3,.3)*self_t4
"

dat   <- simulateData(pop_mod, model.type = "sem", sample.nobs = c(samplesize/2,samplesize/2), group.label = c("no_fb","fb"))

To prevent that participants explicitly remember and integrate the feedback several times, they are randomly assigned into three groups. The groups are re-assessed either after two, four, or six months. This randomization is simulated via:

dat$condition <- sample(1:3,nrow(dat),replace = T)
#set self-estimates to missing based on condition
dat$self_t2[dat$condition != 1] <- NA
dat$self_t3[dat$condition != 2] <- NA
dat$self_t4[dat$condition != 3] <- NA

Fitting model

As the decision for feedback is not randomized, we control for selection effects when testing whether feedback increases accuracy immediately afterwards. To do so, we test whether the increase in accuracy is larger in the feedback than in the no-feedback condition. This is done by first calculating the increases via {r} increase_fb := fb_t1 - fb_t0 an then comparing th increases in both groups via {r} diff_increase := increase_fb - increase_nofb.

To test whether the accuracy decreases in the feedback condition from T1 to T4, we compare the measurement occasions via {r} decrease_fb := fb_t4 - fb_t1

fit_mod <- "
self_t0 ~ c(nofb_t0,fb_t0)*iqtest
self_t1 ~ c(nofb_t1,fb_t1)*iqtest
self_t2 ~ c(nofb_t2,fb_t2)*iqtest
self_t3 ~ c(nofb_t3,fb_t3)*iqtest
self_t4 ~ c(nofb_t4,fb_t4)*iqtest

increase_fb   := fb_t1 - fb_t0
increase_nofb := nofb_t1 - nofb_t0

diff_increase := increase_fb - increase_nofb

decrease_fb := fb_t4 - fb_t1
"

Power Simulation

Data are simulated for samplesizes from 100 to 1000 in 100 increments, with 500 replications for each samplesize. Nonconvergencies are saved in a logfile.

samplesize <- seq(100,1000,100)
nrep <- 500
logfile <- data.frame(matrix(ncol = 3, nrow = 0, dimnames=list(NULL,c('rep','N','message'))))

Simulation Code:

result_incr <- NULL
result_decr <- NULL
for (rep in 1:nrep) {
  for (i in samplesize) {
    errorhandling <- tryCatch({  
      dat   <- simulateData(pop_mod, model.type = "sem", sample.nobs = c(i/2,i/2), group.label = c("no_fb","fb"))
      
      dat$condition <- sample(1:3,nrow(dat),replace = T)
      dat$self_t2[dat$condition == 1] <- NA
      dat$self_t3[dat$condition == 2] <- NA
      dat$self_t4[dat$condition == 3] <- NA
      
      mod <- sem(fit_mod, data = dat, group = "group", missing = "fiml")
    },
    warning = function(w){w},error = function(e){e})
    
    if(inherits(errorhandling,c("warning","error"))){
      logfile[nrow(logfile)+1,] <- c(rep,i,as.character(errorhandling))
    } else{
      par_temp <- parameterEstimates(mod, standardized = T)
      par_temp$rep <- rep
      par_temp$N <- i
      result_incr <- rbind(result_incr,par_temp[par_temp$lhs == "diff_increase",])
      result_decr <- rbind(result_decr,par_temp[par_temp$lhs == "decrease_fb",])
    }
  }
}

SIM and Self-Insight over time

The power simulation pertaining to the longitudinal interplay of the SIM and subjectively perceived self-insigth is based on the random intercept cross-lagged panel model described by Hamaker et al. (2015). An easy to apply r-package ``{r} library(powRICLPM) is available to simulate the power for each of the model’s components (Mulder, 2023).

We based the simulation on within-person cross-lagged effects of .15 and -.15. Data are simulated for samplesizes from 100 to 1000 in 100 increments, again 500 replications each. We added a medium amount of individual differences via an ICC of .60.

Phi <- matrix(c(.4, -.15, .15, .4), ncol = 2, byrow = T)

out <- powRICLPM(
  target_power = 0.8,
  search_lower = 100,
  search_upper = 1000,
  search_step  = 100,
  time_points  = 4,
  ICC          = 0.60,
  RI_cor       = 0.00,
  Phi          = Phi,
  within_cor   = 0.00,
  reps         = 500,
  seed         = 123456
)
